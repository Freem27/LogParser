Для работы небходим .net8 SDK
x64 https://dotnet.microsoft.com/en-us/download/dotnet/thank-you/sdk-8.0.409-windows-x64-installer
x86 https://dotnet.microsoft.com/en-us/download/dotnet/thank-you/sdk-8.0.409-windows-x86-installer

Перетащить текстовый файл логов на logParser.exe
рядом с лог файлом будет создан .csv файл с таким же названием. 
ВНИМАНИЕ если такой .csv файл уже существует он будет презаписан без подтверждения.
